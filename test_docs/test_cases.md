## TC-PR-004 — Получение товара по существующему ID

| Поле | Значение |
|---|---|
| **Модуль** | Товары |
| **Приоритет** | High |
| **Тип** | Позитивный |
| **Предусловия** | Сервис запущен, существует товар `product_id=1` |

**Шаги:**
1. Отправить `GET /products/1`

**Тестовые данные:** `product_id=1`

**Ожидаемый результат:** код 200; тело — карточка товара по схеме `ProductOut` (поля `id`, `name`, `price`)

## TC-OR-001 — Создание заказа с валидными данными

| Поле | Значение |
|---|---|
| **Модуль** | Заказы |
| **Приоритет** | High |
| **Тип** | Позитивный |
| **Предусловия** | Существуют `user_id=1` и `product_id=3` |

**Шаги:**
1. Отправить `POST /orders` с валидным телом

**Тестовые данные:**

